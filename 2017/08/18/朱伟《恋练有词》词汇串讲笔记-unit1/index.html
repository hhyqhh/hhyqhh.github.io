朱伟《恋练有词》词汇串讲笔记(unit1)

发表于 2017-08-18 | 分类于 英语 | 本文总阅读量 次

回想备战考研已经是一年前的事情了。这几天又把《练恋有词》的视频课回顾了一下。（准备考研的小伙伴应该都熟悉这本书吧）。视频的最后10节课是词根词汇的串讲。干货十足，听课之余也做了相应的笔记。

unit 1

引入

unparalleled 无与伦比的

  • un = 否定前缀
  • Para = 部分 偏
    • paralympics 残奥会
  • llel，lel，level 线条
  • -ed (adj)=没有意义

单词结构

  • 前缀 （Prefix）
    • 情感（emotion）
    • 方向（orientation）
  • 词根 （root）
    • 本源 （substance）
  • 后缀 （suffix）
    • 词性(function)

Prefix (情感+方向)

sup- 向上

  • superman ：超人

hyp- 向下

  • hypnosis 催眠
  • hypocrisy 虚伪

post- 向后

  • postpone 推迟
  • post-graduate 研究生

pre- 向前

  • prepaid 提前支付的
  • preterm 早产
  • predetermine 预定

mal- 恶意的（向下的）

  • malice 恶意
  • malicious(adj.): 恶意的
    • be malicious to

Root：(本源 =substance)

Star-

=astr,aster,astro
位置改变，加入元音。

  • 天文
    • astronaut
    • astronomy
  • 运势
    • disaster 倒霉
    • catastrophe 大灾难（cat=向下）
    • cat 向下：catalogu （目录）

Sen-

senior 老者；老迈（老的有层次）

  • senility 老迈
  • seniority 资历
  • seneschal 佣人 管家
    • (-al人)

Suffix：(词性)

-ive =抽象形容词

  • preservative 有保护性的

-ity = 抽象名词

  • persionality ：个性
  • productivity ：生产力

-ate = 厚道之人

  • intimate->intimacy :亲密伙伴->亲密

-or/-er = initiati (主动性的)

-ee = passive （被动性的）

  • employee 雇员
  • refugee 难民

-ist = 专家

  • physicist 物理学家
  • specialist 专家

-al =专业人士

  • professional

-ian =在某些方面执着并且有兴趣的人

  • politician 政客
  • statesman 政治家

Suffix: (单词含义)

-ard 贱人

  • 碰撞
    • ardent （adj）激烈的 剧烈的
    • billiards 碰撞
  • 贱人
    • bastard 王八蛋
    • niggard 吝啬鬼
    • sluggard 懒虫
    • coward 懦夫

-itis = 病（炎症）

  • larynx->laryngitis: 咽喉->咽喉炎
  • hep -> hepatitis : 肝->肝炎
  • appendix -> appendicitis ：附录->阑尾炎
  • diabetes 糖尿病

-ia 疾病

  • via = 通过
    • media 媒体
    • aviation 航空
      • aviator 名航飞行员
      • pilot 试飞员
  • 重大疾病
    • 安乐死
      • euthanasia
      • mercy killing （仁慈杀死）
    • 抑郁症/多疑症状
      • hypochondria
        • chon 长期的：chronic disease 慢性病
      • phobia

原则

原则一 ：顺序改变，意思不变

  • mature （n.）->amateur（n.）:成熟->业余 （a- 反义）
  • center 中心
    • centri->eccentric：离心的，古怪的

原则二 ：元音替换，意思不变(a,e,i,o,u,y)

  • symmetry 相称
    • sym = same = sem =sim
    • simulate 模拟
      • metry = meter ：米->测量工具
      • thermometer：温度计
    • be sympathetic to 对…同情
    • empathy 同理心
    • symmetry <–> asymmetry :相称的/不相称的
    • sympathetic <–> asympathetic

原则三 ：发音相近，意思也相近

  • dialogue = dia 对面 + log（说话 = loq = log）
  • diagnose = 诊断
  • eloquence ：e（e = ex= out）+loq
    口才，雄辩。
    • Facts speak louder than eloquence :事实胜于雄辩。

前缀-Prefix （向上的）：

up-：

  • uphold(v.):支持
    • share holder :股东
    • household :家庭主妇
  • upsurge : 猛增
    • surge : 增加 增长
  • uproar : 大声喧嚣
    • roar : 咆哮 喧嚣
    • disturbance : 心烦意乱
    • riot : 骚乱

sup-;sus-:

  • sustain: 坚持
    • tain=stay/catch/hold:抓住、留下
      • tain：obtain one’s approval:得到某人的同意
      • detain: 拘留
      • maintain : 维持 维修 主张（set forth）
        • set forth a family 养家糊口
          • make both ends meet
          • income <–> expenditure / expense
          • outcome 结果结局
            • disastrous outcome/consequence 灾难性的后果
          • breadwinner 持家者
            • Who wins the bread in this family?
        • vehicle-maintenance :汽车保养
    • sustainable development：可持续发展

sup-;super:(过犹不及)

  • superior （a/n）:上属，优等
    *<–>inferior:下属；下等的。
    • superiority 优越性
    • inferiority 劣根性

suf- ，sur-

  • on-line surfing :在线冲浪
    • suf- ：补充的意思
      • sufficient（a）充足的 大量的
      • sufficiency (n.) 充足 大量
      • suffice sb in ：在某些方面帮助某人
    • deficient 匮乏；insufficient 不足的。前者程度大于后者。
    • efficient 效率高的
      • energy-efficient house节能房
      • energy-saving house 节能房

over-：a/b

  • 在上面的：overlap(v.) 重叠
    • lap
    • lapse下降 瑕疵
    • collapse 崩溃
    • eclipse 月食 没落
  • 过
    • Bite off（咬走） more than one can chew（咀嚼） 过犹不及
  • over-sensitive :过度敏感
  • over-insistence（忍，坚持）:偏执
  • overwhelm(v.)淹没
    • drown （过程）= sink (结果) 下沉
    • oblivion 湮灭
    • The flood overwhelmed the whole village
    • An overwhelming number of… 大量的
    • immerse=浸在…; 沉浸在…
      • marry A with B
      • be immersed in sth.融入做某事
      • abandon oneself to grief/pleasure (沉静在痛苦/快乐中)。
      • M&A : Merger and Acquisition 合并和兼并
      • acquire ->acquisitive(a.)贪婪
        • greedy 贪婪的
      • merse = merge = 吞并
        • submerge：sub= 下面
        • be submerged in… = be engaged in… :沉浸其中。
        • saturate= 饱和 satisfy满意

hyper-:过

  • hypersensitive (a) 敏感的
    • hypersensitive skin :过敏性皮肤
  • hypercritical（a）:吹毛求疵的
    • hypocritical (): 虚伪的
    • critic 评论家；critique 社评 评论
    • skeptic：怀疑论者
      • skim 跳读 略读
      • scan 扫描
      • be skeptical of 对什么表示怀疑的
      • critical：至关重要的
        • it is critical that
